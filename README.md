# Лабораторные работы и практические занятия по курсу "Алгоритмы и структуры данных"

## Данный репозиторий содержит те задания, которые написаны на языке `csharp`.


## Запуск:
1. Склонировать репозиторий
2. Открыть его в Rider
3. Перейти в нужный `project`, соответствующий искомой задаче
4. Ознакомиться с кодом
5. Нажать на кнопку Run, которая соберёт и запустит проект.
